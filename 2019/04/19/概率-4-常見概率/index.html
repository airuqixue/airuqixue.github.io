概率-古典概率

发表于 2019-04-19 更新于 2023-02-07 分类于 概率

生活中的一些例子，可以归纳出一些概率求取方法。
叶老师的“数数算几率”

古典概率

古典概率假设每个实验结果发生概率相同。

Ex 包子摊肉包，菜包，豆沙包产量相同，外表一致。

P(买到咸包子) = 1/3 * 2

计算某事件概率的问题，等同于计算次事件包含了多少实验结果。
所以把计算概率问题转换成了数数问题。

数数基本原则(Fundamental Principle of Counting)

若某种实验有n种不同结果， 而另外一种实验有m种不同结果。 若操作此两实验将有nm种不同结果。

Ex 下午茶有 5 中甜点可选，10 种饮料可选，一共有多少种下午茶组合？
5 x 10 = 50
n m nm

所有物件是否可区分 (Distinguishable?)

实验中选取的物件是否可放回共下次抽选(With/Without Replacement?)

实验中被选取的东西，是否顺序相关 ？(Order matter or not)

例子

排列(Permutation)

EX 小美周日购物，从小明，小华，小袁三兄弟接送。若这两日司机不可重复，问有多少种结果？

  • 可区分？ 是
  • 可放回？ 否
  • 顺序有差异？ 是

所以

一般化, 若有 n 个不同物体, 依次取出 k 个物体共有多少种结果?

$n \times (n-1)\times (n-2)\times …\times (n-(k-1)) = \frac{n!}{(n-k)!}$

重复选取(Choose with Replacements)

Ex 小美周日购物，找小明，小华，小袁接送。若不排除连续两天同一个人，请问多种结果

  • 可区分？ 是
  • 可放回？ 是
  • 顺序有差异？ 是

一般化 若有 n 物体，从中选取一个，每次取完放回，依次取k次，共有多少中结果?

$n \times n \times n \times … \times n = n^k$

组合(Combination)

Ex 小美玩跳棋。从明华袁三兄弟找两人下棋，请问多少种组合对战

  • 可区分？ 是
  • 可放回？ 否
  • 顺序有差异？ 否

$\frac{3 \times 2}{2!} = \tbinom{3}{2} = 3$

一般化 若有 n 个物体，从中选取出 k 个， 共有多少重结果

$n\times (n-1)\times (n-2)\times … \times (n-(k-1))/k! = \frac{n!}{(n-k)k!}$

$\tbinom{n}{k} (binomial — coefficients) 二项式系数$
二项式定理 $(x+y)^n = \sum_{k=0}^n\tbinom{n}{k}x^ky^{n-k}$

Ex 某个球队有2个篮球队员，问有多少种先发组合？

$\tbinom{12}{5} = \frac{12!}{5!7!} = 792 种组合$

多项组合(Multinomial)

Ex 废文兄习惯在网络上发废文（水贴），第一楼回帖常有4类

  • 1 你妈知道你在发废文吗？
  • 2 见此ID必嘘
  • 3 在五楼……
  • 4 妈！我在这！

问， 废文兄发10次废文，一楼回帖共有多少中组合？ $4\times 4\times 4\times…\times 4 = 4^{10}$
问， 多少组合会看到 4次 【你……】， 3次【见……】，2次【在五楼……】， 1次【妈……】

—— —— —— —— —— —— —— —— —— —— 第一次10中选4 ，第二次只剩下 6中选3，第三次省3中选2， 第四次1中选1

所以

$\tbinom{10}{4} \tbinom{6}{3} \tbinom{3}{2} \tbinom{1}{1} = \frac{10!}{4!3!2!1!}$

一般化 若有 m 种不同的物体，每次选取一个放回，依顺序选取n次。 如此共有 $m^n$ 种结果。 其中第 1 种出现 $\lbrace n_1\rbrace$ 次，
且第 2 种出现 $\lbrace n_2\rbrace$ 次， 且第 m 种出现 $\lbrace n_m\rbrace$ 次，这样的实验结果有多少种。

$\tbinom{n}{n_1} \tbinom{n-n_1}{n_2} \tbinom{n_m}{n_m} = \frac{n!}{n_1!n_2!…n_m!}$

Multinomial coefficient $\frac{n!}{n_1!n_2!…n_m!}$

古典概率数数算概率

若一事件包含数个实验结果，每个实验结果发生的概率一样那么

  • 1 先计算一个实验结果的概率
  • 2 再计算该事件共包含多少个实验结果

费文兄发废文例子，根据统计，费文兄发文的回帖出现概率如下

  • P(你妈知道你在发废文吗) = 0.4
  • P(见此ID必嘘) = 0.2
  • P(在五楼……) = 0.1
  • P(妈！我在这！) = 0.3

Ex 若费文兄发文6次，会在一楼回帖看到 2次 【你……】， 2次【见……】，1次【在五楼……】，1次【妈……】的概率为？

P(你 你 见 见 五 妈) = 0.4 x 0.4 x 0.2 x 0.2 x 0.1 x 0.3 = 0.000192

这只是一种组合！！！！！ 所以再算算有多少这样的组合？

$P(2你，2见，1五，1妈) = \frac{6!}{2!2!1!1!} . 0.000192 = 0.0346$

这就是古典概率的好处。
