概率-随机变量

发表于 2019-04-22 | 分类于 概率

随机变量是什么？ 回忆废文兄发废文，我们要这么写
P(“你妈知道你在发废文吗”)=0.4
P(“见此ID必嘘”) = 0.2
P(“在五楼……”) = 0.1
P(“妈！我在这”) = 0.3

P(“妈！我在这”) = 1 - P(“你妈知道你在发废文吗”) - P(“见此ID必嘘”) - P(“在五楼……”)

有没有觉得有问题？ 那就是太啰嗦了！

随机变量

如果你会编程，那么就太好理解。我们把这几种情况定义为一个变量 X，那么

X=0 表示 “你妈知道你在发废文吗”
X=1 表示 “见此ID比嘘”
X=2 表示 “在五楼……”
X=3 表示 “妈！我在这！”

根据统计

P(X=0)=0.4
P(X=1)=0.2
P(X=2)=0.1
P(X=3)=0.3

P(X=3) = 1 - p(X=0) = p(X=1) - P(X=2)

清爽了许多吧。

随机变量(Random Variable, R.V.)： 是用来把实验结果数字化的表示方式
X就是随机变量， 一般都用大写表示

本质

随机变量是一个函数 ？

X(“你妈知道你在发废吗”) = 0

有一个对应关系。 X: S -> R

从样本空间到实数的映射

随机变量的种类

  • 离散随机变量 (Discrete R.V.)

Ex - 店员微笑 X(微笑) = 0， X(不笑) = 1
X=0， X=1

离散R.V.的值是有限个，或者是“可数的” 无穷多个

  • 连续随机变量(Continuous R.V.)
    Ex 幸运之轮: X 可以是 0 到 1间内的任意数字

连续R.V.的值是无穷多个，而且是“不可数”的无穷多个

可数与不可数

可数

Ex 正偶数集合 {2,4,6,8,10,…} 这个是可数的么？

Yes， 这是一个一个可数的，虽然是无穷多个。

不可数

就算你可以不停地数，也有数不到的数！

Ex 0 到 1 之间的所有数字的集合是不可数的。

假设存在一种算法，可以把0到1之间的数字列出来，那么，我们可以这样构造一个数字，不在列出来的数字之中！
该算法如下：
小数点后第一位数字，等于9减去列出来的小数点后第一位数字
小数点后第二位数字，等于9减去列出来的小数点后第二位数字
…
小数点后第N为数字，等于9减去列出来的小数点后第N为数字

这个数字与列出来的任何一个数字都不相等，所以可以证明假设不存在！

有趣的事

“正整数的集合” 与 “正偶整数的集合” 的集合，那个集合大？ 一样大
“长度为一线段上的点” 与 “边长为一的正方形上的点” 这两个集合哪个点数多 ？ 一样多

why ？
一一对应。居然可以找到一一对应的关系！！！

这些因为是不可数的集合，其实比较多少是无意义的。

随机变量函数

Ex 阿宅看到店员微笑，点$200的套餐，如果店员不笑，就买$15的饮料，请问阿宅消费金额W是随机变量吗？

X(微笑) = 0, X(不笑)=1
W(X(微笑)) = 200, W(X(不笑)) = 15
so, W 也是一个随机变量
随机变量的函数，也是一个随机变量。
