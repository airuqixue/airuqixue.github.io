概率-条件概率

发表于 2019-04-18 | 分类于 概率

生活中常常遇到很多事情已经发生，那么这些发生的事情对我们的决策有多少影响呢？
我们引入了条件概率。

条件概率( Conditional Probability)

表示为 P(X|Y)

X: 所关心的事件
Y: 条件（观察到的，已经发生的事件)

叶老师的例子，“卷哥漏曲”很有意思，观察到的事件，改变了样本空间可能性。所以在这个条件下，概率也不一样了。

扩展： 若某个实验结果 Oi 与某条件Y不相交， 则

P(Oi|Y) = 0

这也说明，条件被观察到已经发生，而Oi结果已经被排除，所以为0

扩展： 若某条件事件Y包含的实验结果为

$Y = {O_1,O_2,…,O_n}$

$P(O_i|Y) = \frac {P(O_i)}{P(O_1)+P(O_2)+ … +P(O_n)} = \frac {P(O_i)}{P(Y)}$

Ex: 考虑某事件 $ X = \lbrace O_1, O_2, q_1, q_2 \rbrace $, 已知条件事件 $ Y=\lbrace O_1, O_2,O_3 \rbrace $ 发生了，则

$ P(X|Y) = P(O_1|Y) + P(O_2|Y) = \frac{P(O_1)}{P(Y)} + \frac {P(O_2)}{P(Y)} = \frac {P({O_1, O_2})}{P(Y)} = \frac {P(X\cap Y)}{P(Y)}$

Final : 若已知某条件事件Y发生了， 则对于任何事件X， 我们可以计算其条件概率如下

$P(X|Y) = \frac {P(X\cap Y)}{P(Y)}$

如果遇到 condition on, suppose, if, assuming, given that 这些单词，多半就是条件概率了。（考试党福利）

条件概率性质

性质1

$P(X|Y)=\frac{P(X\cap Y)\ge 0}{P(Y)\ge 0} \ge 0$

性质2

$P(Y|Y) = \frac{P(Y\cap Y)}{P(Y)}=\frac{P(Y)}{P(Y)} = 1$

性质3

$A,B 互斥 \Rightarrow P(A\cup B|Y) = \frac{P(A)}{P(Y)} + \frac{P(B)}{P(Y)} = P(A|Y) + P(B|Y)$

定理

Total Probability 定理

若 $C_1, C_2, … , C_n 互斥且 C_1\cup C_2\cup … \cup C_n = S $
则对任意事件A，我们有

$P(A) = P(A|C_1)P(C_1) + P(A|C_2)P(C_2) + … + P(A|C_n)P(C_n)$

这个和切面包那个很像哦，我们用条件概率的公式看

$P(A|C) = \frac{P(A\cap C)}{P(C)}$

带入公式，这就是切面包那个定理嘛！ 只是用条件概率代替交集概率！

Bayes’s Rule (贝叶斯定理)

若$C_1, C2,…,C_n 互斥且C_1\cup C_2\cup … \cup C_n = S$
则对任意事件A，我们有

$P(C_j|A) = \frac{P(A|C_j)P(C_j)}{P(A|C_1)P(C_1) + P(A|C_2)P(C_2)+ … + P(A|C_n)P(C_n)}$

A 从关心的事件，变成了条件 ！！
这就是平时关心的事件突然发生了，我们想知道他的条件发生的概率，这就很有用了。

Ex 店员笑和生意有一定关系，我们之前的例子，对于老板来说他关心的是店铺生意，如果店员笑，那么他的生意如何？

已知 P(满) = 1/4 , p(普) = 1/4, P(惨)=1/2
P(笑|满)=1/6, P(笑|普)=2/6 ， P(笑|惨)=3/6

$P(满|笑)=\frac{P(笑\cap 满)}{P(笑)}$
$ =\frac{P(笑|满)P(满)}{P(笑|满)P(满) + P(笑|普)P(普) + P(笑|惨)P(惨)} $
$ = \frac{\frac{1}{6}.\frac{1}{4}}{\frac{1}{6}.\frac{1}{4} + \frac{2}{6}.\frac{1}{4} + \frac{3}{6}.\frac{1}{2}} = \frac{1}{9}$

在举例子
一个学校里有60%男生，40%的女生。男生都穿长裤， 女生一半长裤，一半裙子。现在有以下问题

  • 1 随机遇到一个学生，穿长裤的概率？
  • 2 随机遇到一个学生，穿裙女子的概率？
  • 3 随机遇到一个穿长裤的学生，他是男生的概率？

我们先来看1题，这个很简单Total Probability定理可知

P(长裤) = P(长裤|男)P(男) + P(长裤|女)P(女) = 1*0.6 + 0.5*0.4 = 0.8

第2题，这个其实也可以用Total Probability定理，不穿裙子就穿长裤，所以也可以这样

p(裙子) = 1 - P(长裤) = 0.2
P(裙子) = P(裙子|男)P(男) + P(裙子|女)P(女) = 0*1 + 0.5*0.4 = 0.2 是一样的。

第3题，这个问题正好与前面的反过来了，我们先看到了长裤，再来反推是男女的概率。这里可以用贝叶斯定理

P(男|长裤) = P(长裤|男)*P(男) / (P(长裤|男)p(男) + P(长裤|女)p(女))
= 1*0.6/(1*0.6 + 0.5*0.4) = 0.75

贝叶斯解决的是后验概率的计算，怎么说呢，之前人们从一个已知比例的黑白球袋子里抓出球来，可以计算出抓出黑或白球的几率。
但是，如果我们不知道黑白球的比例，只是闭着眼睛摸出几个球后，对这个黑白球的比例做一个估计。这就是似然估计。

贝叶斯的应用非常广泛，这里理解有限，不再多说。

概率独立性

定义一

若两事件A，B之概率满足

$P(A\cap B) = P(A)\cdot P(B)$

则A，B两事件称为概率上的独立事件

定义二

若两事件A，B之概率满足

$P(A|B) = P(A)$

则A，B两事件称为概率上的独立事件

概率上的独立事件，可以通过观察，统计的数据，分析两事件是否独立。

引申-多事件的独立

若 事件$A_1, A_2,…,A_n$ 满足下列条件，则称此 n 事件独立 (n > 2)

从中任选 m 事件 $ A_{i_1}, A_{i_2}, …,A_{i_m} $ 均满足

$P(A_{i_1}\cap A_{i_2}\cap … \cap A_{i_m}) = P(A_{i_1})P(A_{i_2})…P(A_{i_m}), m = 2, 3,…,n$

Ex: n = 5, $A_1, A_2,…,A_5$

如果5个事件要独立，需要判断的情况有

m=2 时候，$P(A_1\cap A_2) = P(A_1)P(A_2) P(A_1\cap A_3)=P(A_1)P(A_3) … $
m=3 时候, $P(A_1]cap A_2\cap A_3) = P(A_1)P(A_2)P(A_3) … $
m=4 时候, …
m=5 时候, …

必须全部成立。 等式个数组合 为 $\tbinom{n}{2} + \tbinom{n}{3} + … + \tbinom{n}{n}$
