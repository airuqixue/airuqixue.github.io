概率-公理性质

发表于 2019-04-17 更新于 2023-02-07 分类于 概率

数学的基石是公理，公理是不可证明的。
例如线性代数的8条公理 Ex 加法交换律 a + b = b + a
通过公理，我们可以推出很多定理。
如果其他领域能够符合公理，那么之后的推论都可以应用。所以公理非常牛，我们来看看概率的公理

概率三公理（Axioms Of Probability）

三公理

  • 1 公理一:
    对于任何事件 A 而言， P(A) $\geq 0 $

  • 2 公理二:
    P(S) = 1 S 表示样本空间集合

  • 3 公理三:
    事件A1， A2， …… 互斥 $\Rightarrow$ P(A1 U A2 U A3…) = P(A1) + P(A2) + p(A3) + …

性质

Ex 从一副52长扑克牌中抽一张，结果为Ace的几率为？

Ace = {红桃A， 黑桃A，梅花A，方块A}

根据公理三

性质一 互斥集合拆分

若 $ E = {O_1, O_2, O_3 ,…, O_n}$
则 $P(E) = P({O_1})+ P({O_2}) + P({O_3}) +…+P({O_n})$

 {O1},{O2},{O3}互斥，实验结果不能即是这个又是那个。
 所以根据公理三可推出

性质二 空集概率为0

$P(\phi) = 0 $

证明
$ S \cap \phi = \phi \Rightarrow S , \phi $ 互斥
$ S = S \cup \phi $
$ \Rightarrow P(S) = P(S \cup \phi) = P(S) + P(\phi) $
$ \Rightarrow P(\phi) = 0 $

性质三 补集概率

$P(A) = 1 - P(A^c)$

证明
$ A \cap A^c = \phi \Rightarrow A, A^c $ 互斥
$ S = A \cup A^c $
$ P(S) = P(A) + P(A^c) $
$ 1 = P(A) + P(A^c) $
$ P(A) = 1 - P(A^c) $

性质四 差集概率

$P(A) = P(A-B) + P(A \cap B)$

性质五 交集概率

$P(A \cup B) = P(A) + P(B) - P(A \cap B)$

性质四和五可以用集合图解来很好的证明。

性质六 并集概率

“切面包”

若 $C_1, C_2, …, C_n $ 互斥 切 $C_1 \cup C_2 \cup…\cup C_n = S $
则对任何事件 A： $P(A) = P(A\cap C_1) + P(A\cap C_2)+…+P(A\cap C_n)$

S 就是一个面包，A这个事件的概率就等于A与每一个面包块交集的概率相加。很形象。之后用的很多。

这是一个很有意思的性质。来看叶老师的例子（简化）。
Ex ： 通过对女店员的态度与店里的生意好坏的统计发现一定规律。店里的生意分为满客(F)，普(C)，惨淡(P)， 店员的态度有笑(L)怒(A)
根据统计

现在我们要知道店员笑的概率。 这里运用“切面包”定理来看， 店铺的生意分为 满，普，惨三种，他们的概率加起来应该是1，也就是以上矩阵中所有的可能加起来。
说明店铺生意可以被分成三种（切割成3块）
而店员笑这个事件的概率，就是她与这几种店铺状态的集合交集。

$P(笑)=\frac{1}{20} + \frac{5}{20} + \frac{5}{20} = \frac{11}{20}$

性质七 子集概率

若 $ A\subset B$ ， 则 $P(A) < P(B)$

证明

$ \because A\subset B $
$ \therefore A\cap B = A , P(B-A) > 0 $
$ \because P(B) = P(B-A) + P(A \cap B) $ 性质4变形
$ \therefore P(B) = P(A) + P(B-A) $
$ \because P(B-A) > 0 $
$ \therefore P(A) < P(B) $
