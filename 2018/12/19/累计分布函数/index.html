累积分布函数CDF

发表于 2018-12-19 更新于 2019-04-25 分类于 概率

什么是累计分布函数？他有什么用呢？
上一节讲了随机变量，这里我们继续沿用这个概念来说明累积分布函数。

定义

对于任意一个随机变量 X ， 我们定义其CDF微函数：

这里大写X标识随机变数

怎么理解呢， 还是用幸运之轮的例子

$F_X(0.5) = P(X\leq0.5)=1/2$

这样就好理解了吧。也就是随机变量小于0.5的这样一个概率是多少呢？很明显 ，1/2
（幸运之轮是从0到1的一个圆盘，转到的数值在 [0,1)之间）

用途

当我们需要计算 X 落在某个范围内的概率时候，

$P(-\infty—-3=====5——\infty)$ 3-5之间的范围

$P(3<X\leq5) = P(-\infty<X\leq5) - P(-\infty<X\leq3)$
= $F_X(5)-F_X(3)$

分类

离散随机变数的累积分布函数

来看掷骰子，一共有6个面。(公平骰子哦)
那么 P(X=1) = P(X=2) = P(X=3) = P(X=4)
= P(X=5) = P(X=6)=1/6
CDF: $F_X(x)=P(X\leq x)$

当x=0.1的时候，不可能有骰子点数小于0.2， 所以 F(0.2)=0
当x=1 的时候，骰子为1点的概率是1/6， 所以F(1)=1/6
当x=1.5的时候，骰子点数小于1.5的概率也是1/6，只有1，所以F(1.5)=1/6
当x=2.5的时候，骰子点数小于2.5的情况可能是1，或者2， 所以 F(2.5)=2/6
…
当x=6 时候， 骰子点数小于6的概率是1，因为所有的情况都包含了 F(6)=1

当x>6的时候， F(x)=1
这就是离散分布
通过a,b把计算公式整理得到

$P(a<X\leq b)=F_X(b)-F_X(a)$

计算

$P(3<X\leq5)$

=$F_X(5)-F_X(3)=5/6-3/6=2/6$

$P(3<X<5)$

=$P(3<X\leq5^-)$

=$F_X(5^-)-F_X(3)$

=$F_X(5)-P(X=5)-F_X(3)=1/6$

$P(3\leq X<5)$

=$P(3<X\leq5) + P(X=3) - P(X=5)$

=$F_X(5)-F_X(3)+P(X=3)-P(X=5)=2/6+1/6-1/6=2/6$

$P(3\leq X\leq5)$

=$P(3<X\leq5)+P(X=3)$

=$F_X(5)-F_X(3)+P(X=3)$

=$5/6-3/6+1/6 = 3/6$

连续随机变数的累积分布函数

还是使用幸运之轮的例子 ， X为幸运之轮停下来指向的数字 , X 属于 [0,1)

-CDF: $F_X(x) = P(X\leq x)$

$F_X(-0.1) = P(Xleq-0.1) = 0$

$F_X(0.1) = P(0\leq X\leq0.1) = 0.1$

$F_X(0.5) = P(0\leq X\leq0.5) = 0.5$

$F_X(1) = P(0\leq X\leq1)=1$

$P(0.3<X\leq0.5)$
=$F_X(0.5) - F_X(0.3) = 0.5-0.3 = 0.2$

$P(0.3<X<0.5)$
=$P(0.3<X\leq0.5^-)$
=$F_X(0.5^-) - FX(0.3) = 0.5-0.3 = 0.2$

CDF性质

离散随机变数的CDF

$F_X(x^+)= F_X(x)$
$F_X(x^-)=F_X(x) - P(X=x)$

这里的x上标可以理解为比x多一点点，离x差一点点

连续随机变数的CDF

$F_X(x^-)=F_X(x)=F_X(x^+)$

共同性质

  • $F_X(-\infty) = P(X\leq -\infty) = 0 $
  • $F_X(\infty) = P(X\leq\infty)=1$
  • $0\leq F_X(x)\leq1$
